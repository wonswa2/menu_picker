# menu_picker
# 오늘도 네 말이 맞아 

> "아무거나 먹어도 돼"를 믿었다가 낭패를 본 모든 사람들을 위한 메뉴 추천 서비스

## 프로젝트 소개

"아무거나"라고 말하지만 실제로는 이미 먹고 싶은 게 있는 상황, 누구나 한 번쯤 겪어봤을 겁니다.  
이 서비스는 거절 패턴을 분석해 사용자가 원하는 메뉴를 점진적으로 좁혀가는 **유머러스한 메뉴 추천 웹 서비스**입니다.

---

## 주요 기능

### 1. 카테고리 선택 후 메뉴 추천
- 한식 / 중식 / 일식 / 양식 / 분식 / 전체 중 선택
- 선택한 카테고리 내에서 랜덤 메뉴 추천

### 2. 좋아요 / 이건 별로야
- 좋아요 → 해당 메뉴로 최종 결정
- 이건 별로야 → 다음 메뉴 추천

### 3. 거절한 메뉴 제외
- 한 번 거절한 메뉴는 같은 세션에서 다시 추천되지 않음

### 4. 거절 횟수별 유머 멘트
- 거절할수록 점점 지쳐가는 멘트 표시
- 예: "혹시 이미 정해놓은 거 아닌가요?

### 5. 힌트 질문으로 취향 반영
- 2회 거절 시 팝업 자동 등장
- 든든한/가벼운, 국물/국물없는, 매운/안매운 선택

---

## 🛠️ 기술 스택
