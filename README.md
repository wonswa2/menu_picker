# menu_picker

간단한 웹 기반 메뉴 추천 서비스입니다.

## 기능

- 카테고리 선택 후 랜덤 메뉴 추천
- 좋아요 / 이건 별로야 기능
- 거절한 메뉴는 같은 추천 세션에서 다시 추천되지 않음
- 거절 횟수에 따라 유머 메시지 출력
- 2회 이상 거절 시 힌트 질문 제공
- 태그 기반 필터링으로 추천 후보 좁히기

## 실행 방법

1. 이 리포지토리를 로컬로 열거나 web server에 배포합니다.
2. `index.html`을 브라우저에서 엽니다.
